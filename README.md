Akıllı Stok ve Talep Tahmin Sistemi

Bu projeyi, Python ile Nesneye Dayalı Programlama (OOP) eğitimi için geliştirdim. Projede Kaggle’dan alınan satış verilerini kullanarak satış analizi yaptım.

Amaç; Sistem, geçmiş satış verilerine bakarak tahmini talebi hesaplar ve mevcut stok seviyesine göre stok artırımı gerekip gerekmediğini belirtir.

- Pandas kullanılarak satış verilerinin analizi
- NumPy ile basit talep tahmini (hareketli ortalama)
- Matplotlib ile satış grafiklerinin oluşturulması
- Kaggle üzerinden alınan perakende satış verisi
